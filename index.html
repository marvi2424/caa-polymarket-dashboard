Trader concentration on Polymarket

Built from on-chain Polygon data. Counts are per wallet, and one person can hold several. Shows how much each wallet trades, not why.

Gini measures inequality from 0 (all equal) to 1 (one wallet holds everything). It stays near 0.96 in every window, so inequality is consistently extreme. The top-100 share, though, shrinks as the window lengthens (58% early to 16% over the full history), largely because longer windows contain far more wallets. These are wallets, not people.

In the 2024 election window, politics dominated volume with sports second, together over 90 percent. But the Gini stays near 0.97 inside every category, so concentration is structural, not specific to politics.

Each dot is a wallet, placed so that wallets which trade alike sit close together (by volume, number of trades, markets traded, and trade size). Over the full history the wallets fall into two groups: cluster 1 is about 1.0M small, occasional wallets (0.9% of volume), while cluster 0 is about 718k active, high-volume wallets that drive 99.1% of all volume. The split is essentially one of scale and activity: a small set of heavy traders against a long tail of occasional ones.

Each dot is a wallet, placed so that wallets which trade alike sit close together (by volume, number of trades, markets traded, and trade size). In the election window the wallets separate into 6 finer groups by trade size, activity, and how concentrated each wallet's bets are. The heaviest group (cluster 5) is a small set of wallets holding 93.4% of volume; the most common type (cluster 3) is about 18k wallets that each trade little. These are behavioural groupings from on-chain features, not proven trader identities.
